# Campus Events Hub (iOS)

## About
Team-based academic iOS application built using Swift and SwiftUI to manage campus and UAE events.

## Key Features
- Browse campus and UAE events
- View event details
- Create and update events
- Firebase authentication

## My Contributions

## Technologies

- SwiftUI
- Xcode
- Firebase
